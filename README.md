## Cara Menjalankan
1. Clone repo ini.
2. Jalankan `gan_kelompokX.ipynb` di Jupyter/Kaggle (pastikan PyTorch terinstall).
3. Dataset MNIST otomatis download.
4. Hasil gambar di `generated_images/`, model di `models/`.
